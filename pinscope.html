Connect an Arduino Uno Q over USB via SERIAL (Web Serial, Chrome / Edge),
or attach to a board running a WebSocket bridge via WIFI.

Boards must run the PinScope reference firmware (or a compatible JSON-line protocol on 115200 baud).

WIFI CONNECT

Enter the WebSocket URL of an Arduino Uno Q bridge. Format: ws://host:port or wss://host:port.

MQTT CONNECT

Enter the broker's WebSocket URL. Public test brokers: ws://broker.hivemq.com:8000/mqtt, ws://test.mosquitto.org:8080. The browser subscribes to <base>/+/out and publishes commands to <base>/<deviceId>/in.
